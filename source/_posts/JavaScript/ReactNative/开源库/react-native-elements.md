## Icon



### 基本用法


[https://reactnativeelements.com/docs/1.2.0/icon](https://reactnativeelements.com/docs/1.2.0/icon)查阅基本属性和使用

```JavaScript

<Icon
  type="MaterialIcons"
  name="delete"
  size={24}
  color="#D1D1D6"
  onPress={() => console.log('delete')}
/>
```
